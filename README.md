This is a simple, privacy-first web application that helps people understand
their constitutional rights in India.

Users can describe a problem in normal language, and the application explains
which Articles of the Indian Constitution may be relevant, in easy English.
This project is only for legal awareness and education.


## What this application does

- Understands problems written in simple language
- Identifies relevant Articles of the Indian Constitution
- Explains rights without legal or complex terms
- Classifies situations as Low, Medium, or High severity
- Shows confidence scores for suggested articles
- Works without login and does not store user data


## Key Features

- AI-based issue understanding
- Constitutional rights mapping (India)
- Severity classification (Low / Medium / High)
- Explainable AI responses
- Privacy-first and anonymous usage
- Mobile-friendly and responsive design
- Progressive Web App (PWA)


## Technology Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Serverless backend functions
- AI-based reasoning logic


## Ethics and Safety

- This application provides legal awareness only
- It does not give legal advice, verdicts, or punishments
- It does not replace a lawyer or legal authority
- No personal data is collected or stored
